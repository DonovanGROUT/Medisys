# Frontend Medisys

Ce dossier contient l’application SPA développée avec Vue 3, Vite et Tailwind CSS.

## Structure prévue

- `src/components/` : Composants Vue réutilisables
- `src/views/` : Pages principales de l’application
- `src/assets/` : Images, styles, etc.

## Initialisation

1. Création du projet avec Vite
2. Installation de Vue 3 et Tailwind CSS
3. Configuration de base

---

Chaque étape d’évolution du frontend sera documentée ici pour assurer la cohérence et la traçabilité du projet.
